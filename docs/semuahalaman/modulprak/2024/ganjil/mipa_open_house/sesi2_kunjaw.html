Fungsi

Buatlah fungsi jumlahkan yang menerima input n lalu menghitung

\[1 + 2 + 3 + \dots + n = \text{ ?}\]

def jumlahkan(n):
  hasil = 0
  for i in range(1, n+1):
    hasil = hasil + i
  return hasil
print(jumlahkan(3))
6
print(jumlahkan(100))
5050

Fungsi Rekursif

Barisan Fibonacci: 0, 1, 1, 2, 3, 5, 8, …

\[F_0 = 0, \quad F_1 = 1\]

\[F_n = F_{n-1} + F_{n-2}\]

def fib(n):
  if (n == 0):
    return 0
  elif (n == 1):
    return 1
  else:
    return fib(n-1) + fib(n-2)

\[F_{10} = \text{ ?}\]

fib(10)
55

Faktorial:

\[0! = 1\]

\[n! = n * (n-1)!\]

Contoh:

\[\begin{align*} 5! &= 5 * 4! \\ &= 5 * 4 * 3! \\ &= 5 * 4 * 3 * 2! \\ &= 5 * 4 * 3 * 2 * 1! \\ &= 5 * 4 * 3 * 2 * 1 * 0! \\ &= 5 * 4 * 3 * 2 * 1 * 1 \\ &= 120 \end{align*}\]

def fact(n):
  if (n == 0):
    return 1
  else:
    return n * fact(n-1)

\[10! = \text{ ?}\]

fact(10)
