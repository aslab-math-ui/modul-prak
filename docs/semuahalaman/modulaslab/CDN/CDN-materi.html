🌐 Panduan CDN dengan Cloudinary untuk Asisten Lab

Belajar menyimpan gambar, audio, video, dan file praktikum secara efisien menggunakan Cloudinary. Panduan CDN ini cocok untuk asisten laboratorium matematika, statistika, dan aktuaria.
Keywords

panduan asisten laboratorium, belajar Quarto, Quarto, instalasi quarto, quarto untuk asisten lab, cdn cloudinary, cara simpan gambar di cloud, cloudinary untuk praktikum, embed asset dari cloud, panduan cloudinary indonesia

1 🌐 Apa Itu CDN dan Kenapa Penting?

CDN (Content Delivery Network) adalah sistem penyimpanan file eksternal yang memungkinkan akses file lebih cepat dan efisien. Untuk asisten lab, CDN seperti Cloudinary sangat membantu agar:

  • File besar tidak membebani GitHub repo
  • Gambar dan video bisa di-embed ke website tanpa delay
  • Praktikum lebih interaktif dengan asset multimedia

2 ☁️ Daftar & Login Cloudinary

2.1 🔹 1. Daftar Cloudinary

  1. Buka https://cloudinary.com
  2. Klik tombol Sign up for free
  3. Isi email dan password, lalu klik Create Account

📸 Tambahkan tangkapan layar halaman signup Cloudinary
Nama file: images/cloudinary-signup.png

2.2 🔹 2. Akses Dashboard Cloudinary

Setelah login, kamu akan diarahkan ke dashboard. Di sinilah kamu bisa upload dan mengelola semua file praktikum.

📸 Tambahkan gambar dashboard Cloudinary setelah login
Nama file: images/cloudinary-dashboard.png

3 📤 Upload Gambar dan File Praktikum

  1. Klik tombol Media Library
  2. Pilih tombol Upload → Browse
  3. Pilih file dari komputermu (misalnya gambar soal, PDF tugas, audio, dsb.)
  4. Setelah selesai, klik file tersebut untuk mendapatkan link CDN-nya.

📸 Tambahkan gambar proses upload file
Nama file: images/cloudinary-upload.png

4 🔗 Cara Embed Asset dari Cloudinary

4.1 🖼️ Embed Gambar ke Quarto

Gunakan sintaks berikut di Quarto:

![Judul Gambar](https://res.cloudinary.com/akunmu/image/upload/v1234567890/namafile.jpg)

📸 Tambahkan gambar hasil embed di Quarto website Nama file: images/embed-image-quarto.png

5 🎧 Embed Audio/Video ke Website

Gunakan HTML raw di Quarto:

<video controls width="100%">
  <source src="https://res.cloudinary.com/akunmu/video/upload/v1234567890/video.mp4" type="video/mp4">
</video>

atau:

<audio controls>
  <source src="https://res.cloudinary.com/akunmu/video/upload/v1234567890/audio.mp3" type="audio/mpeg">
</audio>

📸 Tambahkan screenshot halaman Quarto dengan video/audio aktif Nama file: images/embed-video-audio.png

6 📝 Tips dan Catatan

  • Gunakan folder/tagging untuk mengelola file sesuai topik (misal: praktikum1, soal-uts, dll)

  • Untuk file yang sangat besar (misal video panjang), sebaiknya gunakan YouTube saja

  • Jangan lupa backup link CDN di spreadsheet agar mudah dicari nanti
