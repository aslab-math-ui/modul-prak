🔧 Panduan Git dan GitHub untuk Kolaborasi Asisten Lab.

Belajar Git dan GitHub dari nol untuk asisten laboratorium. Gunakan Git untuk mengelola versi modul, kolaborasi tim, dan praktik CI CD di RStudio atau VSCode.
Keywords

panduan asisten laboratorium, belajar Quarto, Quarto, quarto untuk asisten lab, git untuk pemula, github untuk asisten laboratorium, git praktikum, panduan git dan github indonesia, version control modul praktikum

1 Mengapa Asisten Lab Butuh Git & GitHub?

Sebagai asisten, kamu mungkin akan:

  • Mengedit modul bareng tim
  • Mengelola revisi tugas
  • Mendistribusikan file praktikum

Di sinilah Git dan GitHub sangat berguna: kontrol versi, backup otomatis, dan kolaborasi jadi lebih rapi.

2 🛠️ Instalasi Git

2.1 🔹 Windows / Mac / Linux

  1. Download dari: https://git-scm.com/downloads
  2. Ikuti wizard install (biarkan pengaturan default)

📸 Tambahkan screenshot saat memilih installer Git
Nama file: images/git-download.png

2.2 🔹 Cek Git di Terminal

git --version

📸 Tampilkan hasil terminal Git versi muncul Nama file: images/git-version.png

3 🧰 Setting Git Pertama Kali

git config --global user.name "Nama Kamu"
git config --global user.email "emailkamu@example.com"

3.1 Ini penting supaya setiap commit tercatat atas nama kamu.

4 💡 Gunakan Git di RStudio & VSCode

4.1 🔸 RStudio

  • Buka Global Options → Git/SVN → Enable version control

  • Restart RStudio

  • Buat proyek baru dari Git

📸 Screenshot pengaturan Git di RStudio Nama file: images/rstudio-git-setting.png

4.2 🔸 VSCode

  • Install ekstensi GitHub & GitLens

  • Clone repo via Terminal / GUI

git clone https://github.com/namakamu/repo-praktikum.git

📸 Screenshot penggunaan Git di VSCode Nama file: images/vscode-git-clone.png

5 🔃 Alur Kerja Git Dasar

git status
git add .
git commit -m "update modul praktikum"
git push origin main
