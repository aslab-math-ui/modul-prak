Halo pembaca! Semoga kalian dalam keadaan sehat dan semangat belajar. Kalau kalian membaca ini, besar kemungkinan kalian adalah Asisten Laboratorium Matematika, Statistika, atau Aktuaria ya?

Jika iya, selamat datang! Blog ini disusun khusus untuk asisten lab pemula di tingkat sarjana di Indonesia agar kalian bisa cepat menyesuaikan diri dalam tugas-tugas asistensi. Di dalamnya akan dibahas panduan teknis serta alat bantu yang umum digunakan, supaya kalian bisa:

  • Membuat dan menyusun modul sendiri
  • Mengelola materi praktikum secara profesional
  • Berkolaborasi dengan tim melalui Git & GitHub
  • Memanfaatkan penyimpanan luar seperti CDN
  • Menyusun laporan formal dengan LaTeX

1 📘 Modul Quarto untuk Asisten Laboratorium Pemula

1.1 Apa itu Quarto?

Quarto adalah platform dokumentasi dan publishing berbasis Markdown yang powerful. Cocok untuk membuat modul praktikum interaktif atau website lab.

1.1.1 🔹 Topik dalam Panduan Quarto

  • Instalasi Quarto di Komputer Lokal
  • Dasar-Dasar Syntax Markdown & YAML di Quarto
  • Cara Membuat Website Praktikum di Quarto
  • Panduan Menyusun Modul Praktikum Interaktif

1.2 Cara Pakai Panduan Quarto Hands-On

Panduan ini didesain modular agar mudah diikuti dan di-update. Bookmark halaman ini dan cek kembali saat ada update ya.

2 Git & GitHub untuk Kolaborasi Asisten Lab

Git dan GitHub sangat membantu untuk pengelolaan versi dokumen dan kerja tim. Terutama kalau kalian mengembangkan modul bersama tim dosen atau sesama asisten.

2.1 🔹 Panduan Git Dasar

  • Instalasi Git di RStudio & VSCode
  • Komit, push, pull, dan branch sederhana
  • Modul Git CI untuk Praktikum

2.2 🔹 Panduan GitHub

  • Membuat repository pribadi & kolaboratif
  • Membuat dokumentasi dengan README
  • Modul GitHub untuk Pemula

3 🌐 CDN (Content Delivery Network) untuk Materi Praktikum

Agar GitHub tetap ringan, file besar seperti video, audio, dan gambar bisa disimpan di luar GitHub melalui CDN.

3.1 🔹 Apa yang Bisa Disimpan di CDN?

  • File gambar dan video untuk dokumentasi
  • File tugas dalam format PDF/ZIP
  • Audio untuk materi presentasi

3.2 🔹 Panduan Spesifik

  • Panduan CDN untuk Materi Pembelajaran
  • Panduan CDN untuk Tugas Praktikum

4 LaTeX untuk Pelaporan Akademik

Sebagai asisten lab, kamu juga akan membuat laporan atau membantu mahasiswa dalam penulisan laporan. LaTeX menjadi alat penting untuk ini.

4.1 🔹 Dasar LaTeX yang Harus Dikuasai

  • Penulisan simbol matematika
  • Format dokumen formal (judul, daftar isi, dll)
  • Kompilasi dokumen offline dan online
