📖 Panduan Membuat Website Quarto dari Nol

🔍 Gambaran Ikhtisar

Website Quarto memudahkan Anda mempublikasikan kumpulan dokumen dalam satu situs web dengan navigasi, opsi rendering, dan gaya visual terpadu. Navigasi dapat berupa navbar, sidebar, atau keduanya. Anda juga dapat mengaktifkan full-text search.

Website Quarto dapat dipublikasikan ke layanan seperti GitHub Pages, Netlify, Posit Connect, atau server statis/intranet. Lihat Publishing Websites.

🚀 Quick Start

  • VS Code
  1. Ctrl+Shift+P → Quarto: Create Project → pilih Website Project.
  2. Tentukan direktori dan nama proyek.
  3. Klik Preview untuk melihat render otomatis.
  • RStudio
  1. File → New Project → Quarto Website.
  2. Tentukan direktori dan opsi (Git, editor Markdown visual).
  3. Klik Render untuk preview.
  • Terminal
  • ⚙️ File Konfigurasi (_quarto.yml)
  • 🔍 Preview & Render
quarto create project website mysite
quarto preview mysite

Preview otomatis terbarui saat file .qmd disimpan.

🔄 Workflow

project:
  type: website

website:
  title: "Modul Quarto"
  navbar:
    left:
      - href: index.qmd
        text: Home
      - about.qmd

format:
  html:
    theme: cosmo
    css: styles.css
    toc: true
  • Preview: quarto preview (VSCode/RStudio).
  • Re-render penuh: quarto render (untuk memastikan semua perubahan global diterapkan).
Important

Jalankan quarto render sebelum deploy setelah mengubah konfigurasi atau CSS.

🌐 Website Navigation

📋 Navbar

website:
  navbar:
    background: primary
    search: true
    left:
      - text: "Home"
        href: index.qmd
      - text: "About"
        href: about.qmd

🗂️ Sidebar

website:
  sidebar:
    style: docked
    search: true
    contents:
      - text: "Introduction"
        href: introduction.qmd
      - section: "Basics"
        contents:
          - basics-1.qmd
          - basics-2.qmd

🏁 Hybrid Navigation & Breadcrumbs

🧩 Hybrid Navigation

website:
  navbar:
    left:
      - text: "Tutorials"
        href: tutorials.qmd
  sidebar:
    - title: "Tutorials"
      contents:
        - tutorial-1.qmd

🔗 Breadcrumbs

website:
  breadcrumbs: false
