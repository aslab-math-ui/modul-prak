🖥️ Persyaratan Perangkat

Diusahakan membawa laptop pribadi dengan software berikut:

  • Metnum: Python (Anaconda atau Google Colab)
  • PDNum: GNU Octave
  • Saindat: Python (Anaconda atau Google Colab)
  • PSD Aktu: Excel, R dan Python

⚠️ Pastikan software terinstal sebelum praktikum!
