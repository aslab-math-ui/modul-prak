Tujuan Pembelajaran

Setelah menyelesaikan modul ini, mahasiswa diharapkan mampu:

  • 1. Menguasai Operasi Dasar dan Manajemen Data di R
    • Melakukan operasi aritmatika dan manipulasi variabel di R
    • Membuat dan memanipulasi vector serta melakukan sampling
    • Mengimport data dari berbagai format (.csv, .xlsx)
    • Menghitung statistik deskriptif (mean, median, modus, variansi, standar deviasi)
  • 2. Memvisualisasikan Data dengan Efektif
    • Membuat visualisasi frekuensi (bar chart, pie chart, histogram)
    • Membuat visualisasi dua variabel (scatter plot, boxplot, line chart)
    • Menggunakan package ggplot2 untuk visualisasi yang lebih fleksibel
    • Memvisualisasikan data time series
  • 3. Bekerja dengan Distribusi Probabilitas
    • Menghitung PMF/PDF, CDF, quantile, dan generate random numbers
    • Menggunakan distribusi diskrit (Bernoulli, Binomial, Poisson, Uniform diskrit)
    • Menggunakan distribusi kontinu (Normal, Uniform kontinu)
    • Memahami package distr dan distrEx untuk manipulasi distribusi
  • 4. Melakukan Analisis Statistika Inferensia
    • Membuat dan menggunakan loop (for loop dan replicate) untuk simulasi
    • Memahami dan menerapkan distribusi sampling (sample mean, distribusi-t)
    • Melakukan taksiran titik dan taksiran interval (confidence intervals)
    • Melakukan pengujian hipotesis (uji-z, uji-t, ANOVA, uji Chi-squared)
  • 5. Menganalisis Hubungan Antar Variabel
    • Menghitung korelasi antar variabel (Pearson, Spearman, Kendall)
    • Memvisualisasikan korelasi dengan heatmap
    • Melakukan regresi linear sederhana
    • Menginterpretasi hasil analisis regresi

1. Dasar-Dasar R dan Manajemen Data

1.1 Operasi Dasar dan Variabel

R adalah bahasa pemrograman yang powerful untuk analisis data. Mari kita mulai dengan operasi dasar:

  • Operasi Aritmatika
  • Variabel
  • Manajemen
R
# Operasi dasar
2 + 3      # Penjumlahan
[1] 5
7 - 5      # Pengurangan
[1] 2
3 * 5      # Perkalian
[1] 15

[1] 0.75
2^3        # Pangkat
[1] 8
2:4        # Sequence
[1] 2 3 4

1.2 Vector dan Sampling

Vector adalah kumpulan data dengan tipe yang sama, merupakan struktur data fundamental di R.

R
# Membuat vector numerik
nilai_ujian <- c(85, 92, 78, 88, 95)
nilai_ujian
[1] 85 92 78 88 95
# Membuat vector karakter
nama_buah <- c("apel", "jeruk", "pisang")
nama_buah
[1] "apel"   "jeruk"  "pisang"
# Mengakses elemen vector
nama_buah[1]        # Elemen pertama
[1] "apel"
nilai_ujian[2:4]    # Elemen ke-2 sampai ke-4
[1] 92 78 88

Enam Tipe Dasar Vector

Ada enam tipe utama, namun empat di antaranya akan Anda gunakan 99% setiap saat.

  • doubles (Numeric)
  • integers (Bilangan Bulat)
  • characters (Teks/String)
  • logicals (Boolean)

Ini adalah tipe default untuk angka di R. Tipe ini bisa menampung angka desimal.

R
ipk <- c(3.55, 4.0, 2.89)
ipk
[1] 3.55 4.00 2.89
class(ipk) # class() memberitahu kita tipe objeknya
[1] "numeric"

Jika Anda yakin hanya akan bekerja dengan bilangan bulat, Anda bisa membuatnya secara eksplisit dengan menambahkan L di belakang angka. Ini menghemat memori.

R
jumlah_sks <- c(24L, 21L, 18L)
jumlah_sks
[1] 24 21 18
class(jumlah_sks)
[1] "integer"

Untuk menyimpan data teks. Selalu diapit oleh tanda kutip " atau '.

R
mata_kuliah <- c("Kalkulus", "Aljabar Linear", "Statistika")
mata_kuliah
[1] "Kalkulus"       "Aljabar Linear" "Statistika"    
class(mata_kuliah)
[1] "character"

Hanya bisa berisi tiga nilai: TRUE, FALSE, atau NA (Not Available/Missing). Sangat penting untuk operasi perbandingan dan filter.

R
lulus_ujian <- c(TRUE, TRUE, FALSE)
lulus_ujian
[1]  TRUE  TRUE FALSE
class(lulus_ujian)
[1] "logical"

Sampling sangat penting untuk analisis statistik:

R
# Sampling tanpa pengembalian (without replacement)
sample(nama_buah, 2)
[1] "jeruk"  "pisang"
# Sampling dengan pengembalian (with replacement)
sample(nama_buah, 5, replace = TRUE)
[1] "jeruk"  "pisang" "pisang" "apel"   "pisang"

3. Distribusi Probabilitas

3.1 Konsep Dasar Distribusi

Untuk setiap distribusi di R, tersedia 4 fungsi utama:

  • d...(): Density function (PMF untuk diskrit, PDF untuk kontinu)
  • p...(): Probability function (CDF - Cumulative Distribution Function)
  • q...(): Quantile function (inverse CDF)
  • r...(): Random number generator (RNG)

  • Uniform Diskrit

# Menggunakan Z-score
