Tujuan Pembelajaran

Setelah menyelesaikan modul ini, mahasiswa diharapkan mampu:

  • 1. Melakukan Operasi Dasar R
    • Menggunakan R untuk operasi aritmatika dan logika.
    • Memahami dan membuat objek dasar di R, yaitu vector.
  • 2. Mengelola Data di RStudio
    • Mengimpor dataset dari file .csv ke dalam R sebagai sebuah data frame.
    • Melakukan inspeksi dasar pada data frame untuk memahami struktur dan isinya.
    • Memfilter (subsetting) data berdasarkan kriteria tertentu.
  • 3. Menganalisis dan Memvisualisasikan Data
    • Menghitung statistik deskriptif dasar (mean, median, min, max, standar deviasi).
    • Membuat tabel frekuensi untuk data kategorik.
    • Membuat visualisasi frekuensi yang sesuai dengan tipe datanya (bar chart, pie chart, histogram).

1. Skenario: Misteri Bunga Iris üïµÔ∏è

Selamat datang kembali! Di modul sebelumnya, kita sudah berhasil menyiapkan ‚Äúbengkel‚Äù kerja kita. Sekarang, saatnya menggunakan alat-alat tersebut untuk memecahkan kasus pertama kita.

Seorang botanis (ahli tumbuhan) datang kepada kita dengan dataset berisi pengukuran 150 bunga Iris. Data ini terdiri dari tiga spesies berbeda. Sang botanis punya beberapa pertanyaan: 1. ‚ÄúApa karakteristik umum dari data yang saya kumpulkan ini?‚Äù 2. ‚ÄúApakah ada spesies yang cenderung memiliki kelopak (sepal) yang lebih panjang dibandingkan yang lain?‚Äù

Tugas kita adalah menggunakan R untuk mengeksplorasi data, menghitung statistik dasar, dan membuat visualisasi untuk menjawab pertanyaan-pertanyaan tersebut.

NoteTentang Dataset iris

Dataset iris adalah data klasik dalam dunia statistika, berisi pengukuran empat fitur (panjang & lebar kelopak dan mahkota bunga) dari tiga spesies Iris. Dataset ini sudah tersedia secara built-in di R, jadi kita bisa langsung menggunakannya.

2. Pemanasan: Operasi Dasar & Vector

Sebelum membedah dataset iris, mari kita hangatkan jari kita dengan beberapa operasi dasar yang akan sering kita gunakan.

2.1. Aritmatika dan Variabel

Seperti yang kita pelajari di modul pertama, R bisa berfungsi sebagai kalkulator. Kita juga bisa menyimpan hasil perhitungan dalam variabel menggunakan <-.

# Operasi dasar
2 * (5 + 3)
[1] 16
# Menyimpan hasil ke variabel
lebar_kelopak <- 2.5
panjang_kelopak <- 4.5
luas_kelopak <- lebar_kelopak * panjang_kelopak
luas_kelopak
[1] 11.25

2.2. Vector: Menyimpan Kumpulan Data

Saat menganalisis data, kita jarang bekerja dengan satu angka saja. Kita akan bekerja dengan sekumpulan data. Di R, cara paling dasar untuk menyimpan sekumpulan data adalah dengan vector, menggunakan fungsi c() (combine).

# Vector berisi data numerik (lebar kelopak dari 3 bunga)
kumpulan_lebar <- c(2.5, 3.1, 2.8)
kumpulan_lebar
[1] 2.5 3.1 2.8
# Vector berisi data teks/karakter (nama spesies)
kumpulan_spesies <- c("setosa", "versicolor", "virginica")
kumpulan_spesies
[1] "setosa"     "versicolor" "virginica"

Setiap kolom dalam sebuah tabel data di R pada dasarnya adalah sebuah vector.

3. Investigasi Data Iris

Sekarang kita siap untuk memulai investigasi. Mari kita muat dan periksa data iris.

3.2. Menjawab Pertanyaan 1: Statistik Deskriptif

Untuk menjawab pertanyaan pertama botanis (‚ÄúApa karakteristik umum dari data ini?‚Äù), kita bisa menggunakan statistik deskriptif.

  • Ringkasan Cepat: Gunakan fungsi summary() untuk mendapatkan ringkasan statistik (Min, Median, Mean, Max, Kuartil) untuk setiap kolom numerik.

  • Perhitungan Spesifik: Anda juga bisa menghitung statistik satu per satu. Untuk mengakses satu kolom, gunakan simbol $.
# Rata-rata panjang kelopak (Sepal.Length)
mean(iris$Sepal.Length)
[1] 5.843333

Ini memberikan kita gambaran umum yang bagus tentang rentang dan pusat data kita.

3.3. Menjawab Pertanyaan 2: Perbandingan Spesies

Sekarang pertanyaan kedua: ‚ÄúApakah ada spesies yang cenderung memiliki kelopak (sepal) yang lebih panjang?‚Äù Untuk menjawab ini, kita perlu memfilter data dan memvisualisasikannya.

Memfilter Data

Kita ingin melihat data hanya untuk spesies tertentu atau untuk bunga dengan kriteria tertentu. Gunakan fungsi subset().

# Membuat data frame baru yang hanya berisi spesies 'virginica'
iris_virginica <- subset(iris, Species == "virginica")

# Melihat 6 baris pertama dari data baru kita

Visualisasi Frekuensi

Cara terbaik untuk membandingkan adalah dengan visualisasi. Mari kita lihat distribusi frekuensi dari data kita.

  • Tabel Frekuensi: Pertama, kita hitung ada berapa bunga untuk setiap spesies.
tabel_spesies <- table(iris$Species)
tabel_spesies

    setosa versicolor  virginica 
        50         50         50
  • Bar Chart: Visualisasi ini paling cocok untuk membandingkan jumlah antar kategori.
barplot(tabel_spesies, 
        main = "Distribusi Spesies Bunga Iris",
        xlab = "Spesies",
        ylab = "Jumlah Bunga",
        col = c("skyblue", "salmon", "lightgreen"))
Dari sini terlihat bahwa jumlah data untuk setiap spesies sama (50).
  • Histogram: Untuk melihat distribusi data numerik seperti Sepal.Length.
hist(iris$Sepal.Length,
     main = "Distribusi Panjang Kelopak (Sepal)",
     xlab = "Panjang Kelopak (cm)",
     ylab = "Frekuensi",
     col = "lavender")
Histogram ini menunjukkan bahwa mayoritas bunga memiliki panjang kelopak antara 5.5 dan 6.5 cm.

Kesimpulan Investigasi

Dengan analisis singkat ini, kita bisa kembali ke sang botanis dengan jawaban: 1. Karakteristik umum data menunjukkan panjang kelopak (Sepal) berkisar antara 4.3 hingga 7.9 cm, dengan rata-rata 5.84 cm. 2. Ya, ada spesies yang cenderung memiliki kelopak lebih panjang. Data menunjukkan bahwa bunga dengan panjang kelopak di atas 7 cm hampir seluruhnya adalah spesies virginica.

Latihan

  1. Hitung rata-rata Petal.Length (panjang mahkota) untuk spesies setosa saja. (Gunakan subset() terlebih dahulu).
  2. Buatlah histogram untuk variabel Petal.Width (lebar mahkota). Beri judul dan warna yang sesuai.

Lampiran: Impor Data dari File Lokal

Di dunia nyata, data tidak selalu ada di dalam R. Seringkali Anda akan mendapatkannya dalam bentuk file .csv atau .xlsx.

  • Impor dari .csv
  • Impor dari .xlsx

Untuk mengimpor file .csv, Anda perlu library readr.

# install.packages("readr") # Jalankan sekali jika belum terinstal
library(readr)

# Misalkan file Iris.csv ada di folder data
data_csv <- read_csv("data/Iris.csv")

Untuk mengimpor file Excel, Anda perlu library readxl.

# install.packages("readxl") # Jalankan sekali jika belum terinstal
library(readxl)

# Misalkan file Iris.xlsx ada di folder data
data_excel <- read_excel("data/Iris.xlsx")
