Pengenalan Dasar

Alur kerja sebuah program jika disingkat adalah terima input, proses data, lalu keluarkan output.

print("Halo dari Departemen Matematika FMIPA UI!")

# Ini komentar dalam kode Python dan tidak akan dieksekusi.
Halo dari Departemen Matematika FMIPA UI!

Anda bisa melihat di atas, program sederhana yang menampilkan teks “Halo dari Departemen Matematika FMIPA UI!” ke layar. Program tersebut hanya memiliki satu baris kode. Perintah print() adalah perintah bawaan di Python yang berfungsi untuk menampilkan sesuatu ke layar. Apa yang ada di dalam tanda kurung dan tanda kutip akan ditampilkan apa adanya.

Operasi Dasar

Operasi dasar matematika seperti penjumlahan, pengurangan, perkalian, dan pembagian juga dapat dilakukan di dalam program. Berikut adalah operator dasar matematika di Python:

  • Penjumlahan: +

  • Pengurangan: -

  • Perkalian: *

  • Pembagian: /

  • Modulus (sisa bagi): %

  • Pemangkatan: **

# operasi tambah dan kurang

a = 10 + 10

b = 5.5 - 4.5

print(a)

print(b)
20
1.0
# operasi perkalian dan pembagian

c = 4 * 5

d = 21 / 4

print(c)

print(d)
20
5.25
# operasi modulus dan pangkat

e = 10 % 3

f = 2 ** 3

print(e)

print(f)
1
8

Operasi dasar ini tidak hanya dapat dilakukan pada bilangan, tetapi juga pada teks (string). Namun, operasi yang dapat dilakukan pada string terbatas pada penjumlahan (penggabungan) dan perkalian (pengulangan).

Latihan

Operasi Perbandingan

Operator perbandingan digunakan untuk membandingkan dua nilai atau data. Hasil dari operasi perbandingan ini adalah nilai boolean, yaitu True (benar) atau False (salah). Berikut adalah beberapa operator perbandingan yang umum digunakan:

  • Sama dengan: ==

  • Tidak sama dengan: !=

  • Lebih besar dari: >

  • Lebih kecil dari: <

  • Lebih besar atau sama dengan: >=

  • Lebih kecil atau sama dengan: <=

# sama dengan

print(5 == 5) 

print(5 == 3)
True
False
# tidak sama dengan

print(5 != 3)

print(5 != 5)
True
False

Tipe Data

Ketika kita akan melakukan berbagai macam operasi tentu kita perlu mengetahui apa yang kita operasikan. Pada contoh sebelumnya sempat disinggung integer, float, dan string. Ketiga hal tersebut adalah tipe data dasar yang sering kita temui di dalam pemrograman.

Secara umum tipe data dapat dibagi menjadi dua kategori besar, yaitu tipe data numerik dan tipe data non-numerik yang meliputi string, karakter, dan boolean. Walaupun begitu masih banyak tipe data lain yang lebih kompleks yang tidak akan kita bahas di sini. Berikut adalah penjelasan singkat mengenai tipe data dasar yang sering kita temui:

  • Integer: Tipe data ini merepresentasikan bilangan bulat, baik positif maupun negatif. Contoh: -3, 0, 42.

  • Float: Tipe data ini merepresentasikan bilangan desimal atau pecahan. Contoh: 3.14, -0.001, 2.0.

  • Character: Tipe data ini merepresentasikan satu karakter tunggal, seperti huruf, angka, atau simbol. Contoh: 'A', '7', '#'.

  • String: Tipe data ini merepresentasikan kumpulan karakter yang membentuk teks. Contoh: "Hello, World!", "12345", "Python is fun!".

  • Boolean: Tipe data ini merepresentasikan nilai kebenaran, yaitu True (benar) atau False (salah). Contoh: True, False.

Untuk mengecek tipe data dari sebuah variabel di Python, kita dapat menggunakan fungsi type().

# integer
x = 10
print(x)
print(type(x))
10
<class 'int'>
# float
pi = 3.14
print(pi)
print(type(pi))
3.14
<class 'float'>
# boolean
is_active = False
print(is_active)
print(type(is_active))
False
<class 'bool'>
# string
nama = "Fasya"
print(nama)
print(type(nama))
Fasya
<class 'str'>

Struktur Kontrol Program

Struktur kontrol program adalah cara kita mengatur alur eksekusi kode berdasarkan kondisi tertentu atau pengulangan. Struktur kontrol yang umum digunakan meliputi percabangan (if-else, match) dan perulangan (for, while). Pada pertemuan kali ini tidak akan dibahas untuk strukturr kontrol Match.

If-Else Statement

Pernyataan if–else digunakan untuk mengambil keputusan dalam program. Program akan memeriksa suatu kondisi logika, lalu menjalankan blok kode tertentu tergantung hasilnya.

# cek apakah bilangan merupakan bilangan genap atau ganjil

num = int(input("Masukkan angka: "))

if num % 2 == 0:
    print("bilangan genap")
else:
    print("bilangan ganjil")
bilangan genap

Penggunaan if-else tidak terpaku untuk dua kondisi saja. Kita juga dapat menambahkan kondisi tambahan menggunakan elif (else if) untuk memeriksa beberapa kondisi secara berurutan.

# Program untuk menentukan grade berdasarkan nilai ujian

num = int(input("Masukkan nilai ujian: "))

if num >= 90:
    print("A")
elif num >= 80:
    print("B")
elif num >= 70:
    print("C")
elif num >= 60:
    print("D")
else:
    print("E")
A
# Pengecekan akses sederhana

password = input("Masukkan password: ")

if password == "openhouse2024":
    print("Akses diterima")
elif password == "admin":
    print("Akses admin diterima")
else:
    print("Akses ditolak")

For Loop

Konsep loop adalah bagaimana sebuah program dapat mengulang suatu blok kode tertentu selama kondisi yang ditentukan terpenuhi. Ada dua jenis loop yang umum digunakan, yaitu for dan while.

Penggunaan for digunakan untuk mengulang perintah dengan menggunakan jumlah iterasi yang ditentukan sebelumnya.

for i in range(5):
    print("Iterasi ke-",i)
Iterasi ke- 0
Iterasi ke- 1
Iterasi ke- 2
Iterasi ke- 3
Iterasi ke- 4

Kode di atas adalah melakukan perintah pengulangan sebanyak 5 kali dan mencetak perintah ke-\(i\) ke layar. Perhatikan bahwa indeks perulangan dimulai dari 0 hingga 4 hal ini karena indeks pada Python dimulai dari 0, sehingga total iterasi adalah 5 kali dimulai dari 0. Catatan, ada bahasa pemrograman lain indeks perulangan mungkin dimulai dari 1.

Bagaimana jika kita ingin mencetak bilangan dari 1 sampai 5? Kita bisa melakukannya sebagai berikut:

for i in range(5):
    print("Iterasi ke-", i+1)

While Loop

Berbeda dengan for yang mengulang berdasarkan jumlah iterasi tertentu, while mengulang blok kode selama kondisi logika tertentu bernilai True. Sederhananya, selama kondisi yang diberikan pada while terpenuhi, maka blok kode di dalamnya akan terus dijalankan.

ptr = 1

while (ptr <= 5):
    print("Nilai :", ptr)
    ptr = ptr + 1

List

Apa itu List ?

Bayangkan kalo kita punya sebuah wadah kontainer atau sebuah kotak panjang yang sepanjang kotak tersebut diberi sekat, pikirkan sebuah kotak penyimpan hot wheels tapi memanjang. Di setiap sekat kita bisa meletakkan hot wheels kita atau benda lain yang mungkin bisa muat ke dalam setiap sekat yang ada di kontainer tersebut. List di Python kurang lebih dapat digambarkan seperti itu, di mana setiap sekat bisa kita isi dengan suatu informasi seperti String, Integer, dan tipe data lain.

List bisa kita pakai ketika kita ingin menyimpan satu atau lebih data, menginginkan untuk meletakkan urutan dari informasi, atau bahkan merepresentasikan obyek yang lebih kompleks secara sederhana.

buah = ["apel", "jeruk", "pisang"]
print(buah[0])
apel
print(buah[1])
jeruk
print(buah[2])
pisang
print(buah)
['apel', 'jeruk', 'pisang']

Fungsi len() berguna untuk mencari panjang dari sebuah list. Maksud panjang di sini adalah banyaknya item yang ada dalam suatu list.

print(len(buah))
3

Kita dapat menggunakan for loop untuk memncetak buah kita lebih cantik dan sekaligus tanpa harus menuliskan kodenya secara manual.

for item in buah:
    print("Saya suka", item)
Saya suka apel
Saya suka jeruk
Saya suka pisang

Misalkan kita ingin mengganti suatu item dari sebuah list, maka kita bisa langsung saja menimpa item yang ada pada list tersebut.

buah[2] = "anggur"
print(buah)
['apel', 'jeruk', 'anggur']

Bisa dilihat pada sebelumnya kita punya "pisang" tapi sekarang ketika kita timpa dengan string baru, list akan diperbarui.

Fungsi dan Rekursi

Salah satu aspek yang juga sangat penting untuk digunakan dalam pemrograman adalah Fungsi. Fungsi di pemrograman bisa dikatakan memiliki kemiripan dengan fungsi yang ada di matematika. Kita dapat memberikan aturan yang menerima satu atau beberapa nilai sebagai input lalu menghasilkan output. Fungsi adalah blok instruksi yang diberi nama dan bisa digunakan berkali-kali.

Misalkan kita ingin membuat fungsi yang menerima input bilangan real dan mengeluarkan output kuadrat dari bilangan input atau kurang lebih adalah fungsi \(f(x) = x^2\) di Python.

def func(x):

    y = x**2

    return y
func(2)
4

Kita juga bisa memberikan sebuah aturan yang menerima panjang dan lebar dari persegi panjang dan memngeluarkan luas dari persegi panjang tersebut.

def cari_luas(panjang, lebar):

    luas = panjang * lebar

    return luas
cari_luas(10, 8)
80

Misalkan kita ingin menjumlahkan \(n\) bilangan pertama.

def jumlahkan(n):
    
    hasil = 0

    for i in range(1, n+1):

        hasil = hasil + i

    return hasil
jumlahkan(100)
5050

Rekursi adalah proses pengulangan sesuatu dengan cara memanggil dirinya sendiri dari versi sebelumnya. Pikirkan ini menjadi sebuah refleksi diri di mana diri kita saat ini adalah produk dari diri kita di masa lalu (kelazzz). Fungsi Rekursif adalah fungsi yang memanggil dirinya sendiri untuk mengeluarkan output.

Contoh paling mudah yang bisa kita pikirkan adalah bilangan fibonacci. Bilangan fibonacci ke-\(n\) adalah jumlahan dari dua bilangan sebelumnya.

$$ fibonacci\

0, 1, 1, 2, 3, 5, 8, 13, 21, $$

atau secara matematis bisa kita tulis sebagai

\[ \begin{gather} F_0 = 0,\, F_1 = 1\\ F_n = F_{n-1} + F_{n-2} \end{gather} \]

def fibonacci(n):

    if (n == 0):
        return 0
    
    elif (n == 1):
        return 1
    
    else:
        return fibonacci(n-1) + fibonacci(n-2)
fibonacci(10)
55

Selalu hati-hati dalam menggunakan fungsi rekursif karena ketika kita tidak mendefinisikan semua kasus yang mungkin terjadi (bisa disebut base case) fungsi akan terus berjalan hingga kompilasi program mengalami error atau kasus terburuknya perangkat kita mengalami crash.

Latihan

Coba buat sebuah fungsi menggunakan rekursi untuk mencari faktorial dari \(n\).

faktorial: \(n! = (n)\times (n-1) \times (n-2) \times \ldots \times 1\)
